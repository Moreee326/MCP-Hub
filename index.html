Features

Everything you need
to manage MCP servers

Add, connect, inspect, and call any MCP server from a beautiful dashboard — no terminal fu required.

🖥️

Visual Dashboard

See all your MCP servers at a glance. Status indicators, tool counts, and one-click connect — no config files to edit.

🔌

One-Click Connect

Connect to any stdio-based MCP server with a single click. Auto-discovers all tools available on the server.

🧰

Inline Tool Calling

Call any MCP tool directly from the dashboard. Fill in arguments via JSON editor, see results instantly — perfect for debugging.

💾

Persistent Config

Server configurations survive restarts. Add once, use forever. Your MCP servers are always ready when you need them.

🔒

Secure by Design

Environment variables for API keys are isolated per server. No credential leakage between MCP processes.

🌙

Dark Mode

Easy on the eyes during late-night coding sessions. Beautiful dark theme with responsive mobile layout.
